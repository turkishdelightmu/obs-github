Backgrounds in CSS are more flexible than they look at first glance. With a handful of properties you can control how a background image scales, tiles, lines up inside its element, and behaves while scrolling. Then, once you’re comfortable, you can swap images for gradients, keep your designs accessible, and even frame images with borders and outlines.

## Background images: the core properties

### 1) `background-image`: set the image

Start by assigning a background image to an element (here, the whole page):

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```


### 2) `background-size`: control scaling (`contain` vs `cover`)

If you want to control the image’s size, use `background-size`.

- **`contain`**: scales the image as large as possible **without cropping** (may leave empty space).
    
- **`cover`**: scales the image to **cover the whole element** (may crop parts of the image).
    

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    min-height: 100px;
  }
</style>
```

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: cover;
    min-height: 100px;
  }
</style>
```

That `min-height` ensures there’s enough vertical space for the background to actually show (useful in small demos or sparse pages).

### 3) `background-repeat`: tile (or don’t)

By default, background images **repeat** to fill the element. You can turn that off or control the direction:

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    min-height: 100px;
  }
</style>
```

Repeat only horizontally:

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-x;
    min-height: 100px;
  }
</style>
```

Repeat only vertically:

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: repeat-y;
    min-height: 100px;
  }
</style>
```

### 4) `background-position`: align the image inside the element

This sets _where_ the background starts. You can use keywords (`top`, `center`, `right`, etc.) or numeric values (percentages/pixels).

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-size: contain;
    background-repeat: no-repeat;
    background-position: center top;
    min-height: 100px;
  }
</style>
```

This centers it horizontally and pins it to the top vertically.

### 5) `background-attachment`: scroll with content or stay fixed

This controls whether the background moves when you scroll.

- **`scroll`** (default): background scrolls with the page
    
- **`fixed`**: background stays “stuck” to the viewport
    

```html
<style>
  body {
    background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
    background-position: center top;
    background-attachment: fixed;
  }
</style>
```

### Bonus: shorthand `background`

You can combine multiple background values in one line:

```html
<style>
  body {
    background: center top fixed
      url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
  }
</style>
```

That example sets the image, position, and attachment in one property.

## Background gradients: image-free backgrounds

Gradients are backgrounds generated by the browser—no image files needed.

### Linear gradients

Linear gradients transition colors along a straight line:

```css
.linear-gradient {
  height: 40vh;
  background: linear-gradient(to right, red, yellow);
}
```

```html
<link rel="stylesheet" href="styles.css" />
<div class="linear-gradient"></div>
```

`to right` makes the gradient run left → right.

### Radial gradients

Radial gradients radiate from a center point outward:

```css
.radial-gradient {
  height: 60vh;
  background: radial-gradient(circle closest-side, red, yellow 50%, green);
}
```

```html
<link rel="stylesheet" href="styles.css" />
<div class="radial-gradient"></div>
```

This creates a circular gradient: red in the center, yellow around the midpoint, then green at the edge.

## Accessibility tips for backgrounds

Backgrounds can easily hurt readability if you’re not careful.

### 1) Keep text contrast high

WCAG recommends at least:

- **4.5:1** contrast for normal text
    
- **3:1** for large text
    

Poor contrast example:

```html
<p style="color: lightgray; background-color: whitesmoke;">
  This is an example of poor contrast.
</p>
```

Good contrast example:

```html
<p style="color: white; background-color: darkslategray;">
  This is an example of good contrast.
</p>
```

### 2) Avoid “busy” backgrounds behind text

Even high-contrast text can be hard to read over detailed images or multi-color gradients. If you must place text on an image, consider adding a semi-transparent overlay behind the text.

### 3) Don’t rely on color alone for meaning

If you use red/green to indicate error/success, also include text labels or icons so color-blind users aren’t left guessing.

### 4) Background audio/video: give control

Auto-playing audio/video can be distracting. Always provide obvious pause/mute controls.

## Borders around images: a few solid options

### 1) `border`: the classic

```css
img {
  border: 2px solid red;
}
```

### 2) Different borders per side

```css
img {
  border-top: 4px solid red;
  border-right: 4px dashed blue;
  border-bottom: 4px dotted green;
  border-left: 4px double purple;
}
```

### 3) `outline`: border-like without affecting layout

```css
img {
  outline: 3px solid gold;
}
```

### 4) Rounded corners with `border-radius`

```css
img {
  border: 2px solid red;
  border-radius: 12px;
}
```
